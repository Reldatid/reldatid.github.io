PROJECTS

Neural Network Analysis Tool

This was my final project at General Assembly. I wanted to create something I might use, and this was something I'd desired in the past. I created a rails database to archive data from running a genetic algorithm, and then used some javascript libraries to graph that data is various ways, including having a visual representation of the neural network itself.

Live Demo || GitHub Repo

Tic-Tac-Toe ad Nauseam

As our first project at General Assembly, we were required to implement tic-tac-toe in the browser. I wanted to push myself and have enjoyed playing a variant called "Tic-Tac-Toe-Ten" with friends. Basically its a more tactical game based off regular tic tac toe where you nest 9 boards in one big board. I took it a step further and allowed the player to decide how many times to nest boards withing boards.(within reason)

Live Demo || GitHub Repo
